title
=====

``iati-activities/iati-activity/result/title``

This is the reference page for the XML element ``title``. 

.. index::
  single: title

Definition
~~~~~~~~~~


A short, human-readable title.


Rules
~~~~~








This element must occur once and only once (within each parent element).
